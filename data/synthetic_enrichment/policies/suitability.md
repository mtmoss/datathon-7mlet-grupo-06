# Politica de Suitability (sintetica, ficticia)

Documento sintetico para o RAG. Nao representa regras reais de instituicao.

- Cliente do segmento "novo" NAO deve receber oferta direta de deposito
  (`oferta_deposito`) como primeira acao; prefira conteudo educativo.
- Ofertas devem respeitar elegibilidade: so braços disponiveis no contexto.
- Decisoes sensiveis exigem revisao humana; o sistema apoia, nao executa.
- Nunca usar atributos protegidos (idade, genero, raca) para discriminar.
